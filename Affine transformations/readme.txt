Данная программа реализована на языке Python с использованием библиотеки PeGame.
Программа реализает афинные пребразования с прямой.
В программе реализованны:
 - Отражение по оси X;
 - Отражение по оси Y;
 - Отражение по осям X и Y одновременно;
 - Поворот плоскости влево;
 - Поворот плоскости вправо;
 - Масштабирование в большую и меньшую сторону в 2 раза.
Рисование происходит методами библиотеки PyGame.
